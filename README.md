# 🖥️ Spring Boot + Thymeleaf Website

This project is a dynamic web application built using **Spring Boot** and **Thymeleaf templates**.  
The website has been successfully deployed with the help of **[Railway.app]**(https://portfolio-production-b4c0.up.railway.app/).

---

## 📌 Project Overview

This website demonstrates how to use:
- Spring Boot for backend logic
- Thymeleaf for rendering dynamic HTML views
- Basic static resources like CSS, JS, and images
- Deployment using BOLD with minimal configuration

---

## 🛠️ Tech Stack

| Layer         | Technology     |
|---------------|----------------|
| Backend       | Spring Boot    |
| Frontend      | Thymeleaf      |
| Template Engine | Thymeleaf   |
| Build Tool    | Maven or Gradle|
| Deployment    | BOLD           |
